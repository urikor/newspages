After install Newspages module will create next: 

- The Newspages content type

- "Alternate Title" and "Admin Comment" fields for the Newspages content type, 
  creates database schema for it
  
- 2 pages with templates in admin and user sections Admin page, User page and 
  set links All The News for admin - in admin part, 
  All The News for users - in user part (Main menu)
  
- "Last News" block ("A listing of 3 last Newspages nodes" on admin block page)

- Creates and sets Anonymous and Authenticated users Permissions for the Newspages Module
